# MCP Contact Manager

An AI-powered contact management system built with **n8n**, **Model Context Protocol (MCP)**, and **Airtable**.

This project exposes an MCP server that enables AI assistants to interact with an Airtable database through standardized tool calling. The workflow allows AI models to create, retrieve, search, and delete contact records using natural language commands.

## Features

- AI-powered contact management
- MCP (Model Context Protocol) integration
- Create new contacts
- Retrieve contact records
- Search contacts with filters
- Delete contact records
- Airtable integration
- n8n workflow automation

## Tech Stack

- n8n
- MCP
- Airtable
- JSON
- REST APIs
